Chapter 1. Introduction to Mobicents JAIN SLEE XDM Client Enabler

The JAIN SLEE XDM Client Enabler allows JAIN SLEE Applications to interact with XDM Servers, hiding the network protocol complexity. The Enabler consists in an SBB which can be used in child relations, with a simple asynchronous interface.

The XDM Server is a functional element used in OMA and 3GPP SIP Presence networks, to manage public user documents, in the XML format. An example of such documents is the user's buddy list. Essentially the XDM Server is an XCAP Server with additional interfaces, such as SIP Event Package XCAP Diff, which allows SIP subscriptions on changes done to documents through XCAP.
