Chapter 1. Introduction to Jopr and Mobicents JAIN SLEE Plugin

Jopr is the JBoss/Red Hat solution for enterprise management which provides administration, monitoring, alerting, operational control and configuration in an enterprise setting with fine-grained security and an advanced extension model.

More information regarding Jopr can be found at the official page [http://www.jboss.org/jopr]

Mobicents JAIN SLEE takes advantage of this full-featured platform, with its custom plugin. This allows complete management, such as deployment and installation of JAIN SLEE Deployable Units, creation and activation of Resource Adaptor Entities, etc. as well as monitoring critical resources, such as the number of Activities present, SBB Entities, etc.

Since Jopr relies on a plugin system, the same instance can be used to administer/monitor Mobicents JAIN SLEE, Mobicents SIP Servlets and Mobicents Media Server, providing a seamless environment for managing the complete suite, including the hosting JBoss AS instances as well.
